运行前：
	没有装openpyxl模块的需要安装，可通过 window cmd ->pip install openpyxl指令安装。如果提示（pip : 无法将“pip”项识别为 cmdlet、函数、脚本文件或可运行程序的名称），
	则是没有设置python环境变量需要设置下。
	可以cmd指令设置，这里要设置2个路径，cmd打开命令行窗口 -》输入这2个路径 path=%path%;python安装目录  以及path=%path%;python安装目录\Scripts   （第二个路径是pip的路径）
               也可以手动在环境变量窗口里设置PYTHON环境变量，网上搜如何设置python环境变量随便找个教程即可
